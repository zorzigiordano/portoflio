# Regole del Progetto (Portfolio)

- **Push su GitHub post-modifiche:** Ricordati sempre di avvisare l'utente di eseguire `git push` (o chiedigli se vuole che lo faccia tu, previa autenticazione se necessaria) dopo ogni modifica importante o corposa. Poiché il sito è collegato a Netlify tramite Continuous Deployment, l'unico modo per pubblicare le modifiche live è spingerle su GitHub.
